Good Vibes Yoga Website

April 2022

In this project, we were assigned to create a new yoga class management website that would allow people to sign up for classes based on level, trainer, and date, and also add classes as a trainer.

Technologies: HTML, JS, Python

This project utilized JavaScript to create a navabr to access the home page, classes page, and adding a class page. Python was implemented for the forms on the website that were used when adding a new class to the site. It was all tied together with the use of HTML and this helped me gain experiences in a number of different technologies.

Feel free to access the website at https://cgi.luddy.indiana.edu/~mleitgeb/I211_project.cgi/ and my Github repository at https://github.iu.edu/mleitgeb/I211_project
