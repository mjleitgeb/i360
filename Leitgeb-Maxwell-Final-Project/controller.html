Arduino Controller Project

January 2023 - February 2023

This was a group project in which we were supposed to build a controller using legos around an arduino board. The buttons of the controller each serve a different purpose on the keyboard such as Shift, Spacebar, Return, etc.

Technologies: Arduino

This was a very interesting project as I did not have much previous work with Arduino and it was fun to expirement with how to construct a controller around an Arduino and breadboard. After the contstruction, we were able to compile code as a team that would run when the buttons are pressed. Along with having an action performed on a computer, we also had LEDs perform actions when the button was pressed.
